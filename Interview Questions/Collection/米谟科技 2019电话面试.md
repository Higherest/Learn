### 6.用二进制表示数字n，计算比特位上的1有多少个。
- 右移：与1位或消除比特位上最低位的1，然后右移去掉最低位。比较容易理解
    ```
    int CountBits(int n)
    {
        int count = 0;
        while(n > 0)
        {
            count += n & 1;
            n >> 1;
        }

        return count;
    }
    ```
- n = n & (n - 1) 可消除比特位上最低位的1
    ```
    int CountBits(int n)
    {
        int count = 0;
        while(n >= 1)
        {
            n &= n - 1;
            count++;
        }

        return count;
    }
    ```
